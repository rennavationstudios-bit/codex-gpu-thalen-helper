# Support

This is an independent community project with no guaranteed support SLA. Open a GitHub issue for reproducible bugs or documentation gaps. Use private vulnerability reporting for security issues.

Before filing a bug, run `thalen-helper doctor` and attach a voluntarily exported redacted diagnostic report if useful. Never attach `auth.json`, tokens, cookies, private prompts/responses, private source, or unredacted user paths.

The project cannot support Codex accounts/billing, ChatGPT login, OpenAI service incidents, Ollama model quality, unsupported GPU drivers, overclocking, modified third-party binaries, or data lost outside the helper's managed files.

Useful issue details include helper version, Windows edition/build, architecture, GPU/driver, model tag, error code, exact safe reproduction, and whether auto-start was accepted.
